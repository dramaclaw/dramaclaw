# Code of Conduct

DramaClaw follows the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its community code of conduct.

By participating in this project &mdash; through issues, pull requests,
discussions, or any other channel &mdash; you agree to uphold it.

## Scope

This code applies to all project spaces: the GitHub repository, Discussions,
Discord and other official chat rooms, and any event or social-media space
where someone is representing the project.

## Reporting

If you witness or experience behaviour that violates this code, please report
it privately to the maintainers:

- **Email:** `conduct@dramaclaw.xxx` <!-- TBD: set final inbox before launch -->

Reports are handled confidentially. We will acknowledge receipt within a
reasonable time frame and work with you on next steps. You will not be
retaliated against for filing a good-faith report.

## Enforcement

Maintainers may take any action they deem appropriate, up to and including
temporary or permanent removal from the project. Enforcement follows the
guidelines in the Contributor Covenant linked above.
